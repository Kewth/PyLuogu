# PyLuogu

在终端，享受 coding 的快乐。

## 声明

本项目实质是一个爬虫，用于 OIer 在洛谷更简单的学习，将本项目用于不良用途，后果自负。

## 安装

```bash
git clone https://www.github.com/Kewth/PyLuogu.git
cd PyLuogu
make
```

## 使用

输入 `PyLuogu` 查看详情。

## 特性

- [x] 终端查看指定题目
- [x] 终端查看任务计划
- [ ] 终端搜索题目
- [ ] 终端提交题目
- [ ] 终端查看题解
- [ ] 终端发射犇犇
- [ ] 终端查看讨论
- [ ] 终端私信
